Easy Integration!

To use this Max Headroom background in your own project, just:

1. Import the component: <script type="module" src="path/to/max-headroom.js"></script> 2. Add it to your HTML: <max-headroom-bg></max-headroom-bg> 3. Style as needed: max-headroom-bg { position: fixed; top: 0; left: 0; width: 100%; height: 100%; z-index: -1; }

The component is fully self-contained and responsive. Move your mouse around to see the interactive effects!
